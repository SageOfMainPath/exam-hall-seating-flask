# Exam Hall Seating Arrangement System (Flask)

A Flask-based web application for managing students/classes and generating an automatic exam hall seating plan.

## Features
- Admin login
- Add / delete classes
- Add / delete students
- Auto seating plan generation
- Basic student registration/login

## Tech Stack
- Python
- Flask
- HTML / Bootstrap
- File-based storage (data/*.txt)

## How to Run (Windows)
```bash
python -m venv venv
venv\Scripts\activate
python -m pip install -r requirements.txt
python app.py
